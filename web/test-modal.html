Tentang SICURAH

ðŸ“‹ Tentang Proyek

SICURAH adalah singkatan dari Sistem Monitoring Curah Hujan dan Kelembaban Tanah. Sistem ini dirancang untuk memantau kondisi lingkungan secara real-time dengan tujuan utama mendeteksi dini potensi terjadinya tanah longsor.

Fitur Utama:

  • Dashboard Real-time: Menampilkan data sensor curah hujan, kelembaban tanah, dan kemiringan tanah secara langsung
  • Data Historis: Analisis tren data sensor dalam bentuk grafik interaktif untuk melihat pola dan perubahan kondisi
  • Peta Lokasi: Visualisasi lokasi-lokasi monitoring untuk melihat sebaran sensor di lapangan
  • Sistem Notifikasi: Peringatan otomatis ketika kondisi mencapai level waspada atau bahaya
  • Status Sistem: Monitoring kesehatan perangkat sensor dan konektivitas sistem

Target Pengguna:

Sistem ini ditujukan untuk pemerintah lokal, badan mitigasi bencana, dan masyarakat di daerah rawan longsor untuk meningkatkan kesiapsiagaan dalam menghadapi bencana alam.
