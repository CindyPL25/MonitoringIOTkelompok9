Panduan Testing API

Klik tombol di bawah untuk menguji setiap endpoint API. Pastikan XAMPP (Apache & MySQL) sudah berjalan dan database sudah diimport.

  • Database harus sudah diimport ke MySQL (sicurah_db)
  • Apache dan MySQL di XAMPP harus aktif
  • Response JSON akan ditampilkan di bawah setiap tombol
  • Hijau = Success, Merah = Error

2. Get History Data

Mengambil semua lokasi node dengan status sensor

4. Get Notifications

Endpoint: /api/get_notifications.php
Mengambil notifikasi/alert sistem

5. Post Sensor Reading

Endpoint: /api/post_reading.php
